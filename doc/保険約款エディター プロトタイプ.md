# 保険約款エディター プロトタイプ

## 概要
保険約款のドラフト作成を支援するWebアプリケーションのプロトタイプです。
AI支援機能と類似箇所検索機能を備えたエディターを提供します。

## 機能
- 📝 Monaco Editorベースの高機能エディター
- 🔍 リアルタイム類似箇所検索
- 🤖 AI支援による条項生成
- 📚 文書管理とバージョン管理
- 🔄 WebSocketによるリアルタイム通信

## 技術スタック
- **フロントエンド**: React + TypeScript + Mantine + Monaco Editor
- **バックエンド**: Node.js + Express + TypeScript
- **データベース**: SQLite
- **AI**: OpenAI API (gpt-4.1)
- **リアルタイム通信**: WebSocket

## セットアップ

### 前提条件
- Node.js 20以上
- npm または yarn

### インストール
```bash
# 依存関係のインストール
npm install

# データベースの初期化
npm run init-db
```

### 環境変数設定（オプション）
```bash
# .envファイルを作成
echo "OPENAI_API_KEY=your_openai_api_key_here" > .env
```

注意: OpenAI APIキーが設定されていない場合、AI機能はダミーレスポンスを返します。

### 開発サーバー起動
```bash
# フロントエンドとバックエンドを同時起動
npm run dev
```

アプリケーションは以下のURLでアクセスできます：
- フロントエンド: http://localhost:5173
- バックエンドAPI: http://localhost:3001

## 使用方法

### 基本操作
1. 左サイドバーから既存の文書を選択するか、「新しい文書」ボタンで新規作成
2. 中央のエディターで約款を編集
3. 右側のパネルで類似箇所検索やAI生成を利用

### 類似検索機能
- エディターでテキストを選択すると自動的に類似箇所を検索
- 右上の検索ボックスで手動検索も可能
- 類似度に応じて色分け表示（高：赤、中：オレンジ、低：緑）

### AI生成機能
- 右下のAIパネルでプロンプトを入力
- 「AI生成」ボタンで条項を自動生成
- 生成されたテキストはエディターに自動挿入

### 文書形式
約款は以下の形式で記述してください：
```
第1条（用語の定義）
この約款において、次の用語の意味は...

第2条（保険金を支払う場合）
当会社は、被保険者が...
```

## API仕様

### 文書管理
- `GET /api/documents` - 文書一覧取得
- `GET /api/documents/:id` - 文書詳細取得
- `POST /api/documents` - 文書作成
- `PUT /api/documents/:id` - 文書更新
- `DELETE /api/documents/:id` - 文書削除

### AI機能
- `POST /api/ai/generate` - コンテンツ生成
- `POST /api/ai/search` - 類似検索

### WebSocket
- `search:similar` - 類似検索リクエスト
- `ai:generate` - AI生成リクエスト

## トラブルシューティング

### よくある問題

1. **WebSocket接続エラー**
   - バックエンドサーバーが起動していることを確認
   - ポート3001が使用可能であることを確認

2. **AI生成が動作しない**
   - OpenAI APIキーが正しく設定されているか確認
   - APIキーが設定されていない場合はダミーレスポンスが返される

3. **データベースエラー**
   - `npm run init-db`でデータベースを再初期化
