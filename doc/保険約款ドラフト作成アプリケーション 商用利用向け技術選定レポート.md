# 保険約款ドラフト作成アプリケーション 商用利用向け技術選定レポート

## エグゼクティブサマリー

本レポートは、商用利用を前提とした保険約款ドラフト作成アプリケーションの技術選定について、VSCode/Clineのライセンス制約を踏まえた独自開発可能な代替技術スタックを提案するものです。

### 主要な調査結果

1. **VSCode/Clineの商用利用制約**
   - VSCode製品：商標権により「Visual Studio Code」名称の商用利用は制限
   - Clineソースコード：Apache 2.0により商用利用は可能だが独自ブランディングが必要

2. **推奨技術スタック**
   - **最優先推奨**：Theia IDE Platform + 独自カスタマイズ
   - **代替案1**：Tauri + Monaco Editor + React
   - **代替案2**：Webアプリケーション（Monaco Editor + Node.js）

3. **開発期間・コスト見積もり**
   - Theia IDEベース：4-6ヶ月、開発コスト低-中
   - Tauri + Monaco：8-10ヶ月、開発コスト中-高
   - 完全独自開発：12-18ヶ月、開発コスト高

## 1. 背景と目的

### 1.1 プロジェクト要件
- 商用利用を前提とした保険約款ドラフト作成アプリケーション
- VSCode + Cline拡張機能のような生成AIとの対話型インターフェース
- エディターでの手動編集とAIリクエストの両方に対応
- 過去の約款ドキュメントを参考にした記載の踏襲機能
- UI上でのVSCode/Cline名称表示の回避

### 1.2 調査目的
- VSCode/Clineの商用利用におけるライセンス制約の明確化
- 独自ブランディング可能な代替技術スタックの特定
- 商用利用に適したライセンス体系の選定
- 開発期間・コスト・リスクの評価

## 2. ライセンス分析結果

### 2.1 VSCodeライセンス状況

#### VSCode製品ライセンス
- **ライセンス**: Microsoft Software License Terms
- **商用利用**: 可能（開発・テスト・社内利用）
- **制約事項**:
  - 「Visual Studio Code」名称の商用利用は商標権により制限
  - ソフトウェアの再配布・販売は禁止
  - スタンドアロン製品としての提供は禁止

#### VSCodeソースコード
- **ライセンス**: MIT License
- **権利**: 商用利用・修正・配布・サブライセンス・販売すべて可能
- **条件**: 著作権表示とライセンス表示の保持のみ
- **商標権**: 「Visual Studio Code」名称使用は別途制限

### 2.2 Clineライセンス状況

#### Clineソースコード
- **ライセンス**: Apache 2.0 License
- **権利**: 商用利用・修正・配布・サブライセンス・特許権許可すべて可能
- **条件**: 著作権表示・ライセンス表示・変更箇所明示・NOTICE保持
- **商標権**: 「Cline」名称使用に特別な制限なし

### 2.3 商用利用リスク評価

#### 高リスク要素
- VSCode製品の直接的な再配布・販売
- 「Visual Studio Code」「VS Code」名称の商用利用
- VSCode拡張機能としての直接販売

#### 低リスク要素
- VSCodeソースコードベースの独自エディター開発（商標権注意）
- Clineソースコードの改変利用（Apache 2.0準拠）
- 完全独自ブランディングでの提供


## 3. 代替技術スタック分析

### 3.1 デスクトップアプリケーション開発技術

#### Tauri（推奨度：最高）
**概要**: Rustベースの軽量デスクトップアプリケーションフレームワーク

**技術仕様**:
- **言語**: Rust（バックエンド）+ Web技術（フロントエンド）
- **ライセンス**: MIT/Apache 2.0
- **バンドルサイズ**: 10-20MB（Electronの1/10）
- **メモリ使用量**: 50-100MB（Electronの1/5）
- **プラットフォーム**: Windows, macOS, Linux, iOS, Android

**商用利用評価**:
- ✅ 完全な商用利用許可
- ✅ 独自ブランディング可能
- ✅ 再配布・販売可能
- ✅ ソースコード保護可能

**開発コスト**:
- 学習コスト: 高（Rust習得必要）
- 開発期間: 8-10ヶ月
- 保守コスト: 中

#### Wails（推奨度：中）
**概要**: Goベースの軽量デスクトップアプリケーションフレームワーク

**技術仕様**:
- **言語**: Go（バックエンド）+ Web技術（フロントエンド）
- **ライセンス**: MIT
- **バンドルサイズ**: 15-30MB
- **開発速度**: 高（Goの生産性）
- **制約**: 一部機能が開発中（マルチウィンドウ等）

**商用利用評価**:
- ✅ 完全な商用利用許可
- ✅ 独自ブランディング可能
- ⚠️ 機能制限あり

**開発コスト**:
- 学習コスト: 中（Go習得必要）
- 開発期間: 6-8ヶ月
- 保守コスト: 低

### 3.2 Webベースエディター技術

#### Monaco Editor（推奨度：最高）
**概要**: VSCodeのコアエディターをWeb向けに抽出したライブラリ

**技術仕様**:
- **開発元**: Microsoft
- **ライセンス**: MIT

- ✅ 独自ブランディング可能
- ✅ カスタマイズ自由度最高
- ✅ VSCode互換性

**統合コスト**:
- 学習コスト: 低
- 統合期間: 2-3ヶ月
- カスタマイズ性: 最高

#### CodeMirror 6（推奨度：高）
**概要**: モジュラー設計の軽量Webエディター

**技術仕様**:
- **ライセンス**: MIT
- **設計**: モジュラー・拡張可能
- **モバイル**: 対応
- **パフォーマンス**: 高

**商用利用評価**:
- ✅ 完全な商用利用許可
- ✅ 高カスタマイズ性
- ⚠️ Monaco Editorより機能少

### 3.3 統合開発環境プラットフォーム

#### Theia IDE Platform（推奨度：最高）
**概要**: Eclipse Foundationによるクラウド・デスクトップ対応IDE開発プラットフォーム

**技術仕様**:

**商用利用評価**:
- ✅ 商用利用フレンドリー（EPL 2.0）
- ✅ 独自ブランディング可能
- ✅ 商用サポート利用可能
- ✅ VSCode代替として実績あり

**開発コスト**:
- 学習コスト: 中（TypeScript）
- 開発期間: 4-6ヶ月
- 保守コスト: 低-中
- 商用サポート: 利用可能
