# VSCode・Clineライセンス調査結果

## VSCode ライセンス状況

### 1. VSCode製品ライセンス
- **ライセンス**: Microsoft Software License Terms
- **商用利用**: 可能（明示的に許可）
- **制約事項**:
  - 「Visual Studio Code」の名称・ブランドはMicrosoft商標
  - ソフトウェアの再配布・販売は禁止
  - スタンドアロン製品として提供することは禁止
  - リバースエンジニアリング・逆コンパイルは禁止

### 2. VSCodeソースコード
- **ライセンス**: MIT License
- **権利**:
  - 商用利用可能
  - 修正・改変可能
  - 配布・再配布可能
  - サブライセンス可能
  - 販売可能
- **条件**:
  - 著作権表示とライセンス表示の保持が必要
  - 「Visual Studio Code」の名称使用は別途商標権の問題

## Cline ライセンス状況

### 1. Clineライセンス
- **ライセンス**: Apache 2.0 License
- **権利**:
  - 商用利用可能
  - 修正・改変可能
  - 配布・再配布可能
  - サブライセンス可能
  - 特許権の明示的許可
- **条件**:
  - 著作権表示とライセンス表示の保持
  - 変更箇所の明示
  - NOTICE ファイルの保持（存在する場合）

### 3. 商用利用における制約

#### ブランディング制約
- **VSCode**: 「Visual Studio Code」名称の使用は商標権侵害の可能性
- **Cline**: 「Cline」名称の使用は特に制約なし（Apache 2.0）
- **推奨**: 独自ブランド名での提供が安全

#### 技術的制約
- **VSCode拡張機能**: VSCode Marketplace経由での配布が一般的
- **独立アプリ**: VSCodeソースコードを基にした独自エディター開発は可能
- **UI表示**: VSCode/Clineの名称をUI上に表示しない独自実装が必要

## 商用利用のリスク評価

### 高リスク
- VSCode製品の再配布・販売
- 「Visual Studio Code」「VS Code」名称の商用利用
- VSCode拡張機能としての直接販売

### 中リスク
- VSCodeソースコードベースの独自エディター（商標権注意）
- Clineソースコードの改変利用（Apache 2.0準拠必要）

### 低リスク
- 完全独自開発のエディター
- オープンソースライブラリの活用
- 独自ブランディングでの提供


## 独自エディター開発技術調査結果

### 1. Electron代替技術

#### Tauri（推奨度：高）
- **言語**: Rust + Web技術
- **ライセンス**: MIT/Apache 2.0
- **特徴**:
  - 軽量（バンドルサイズ小）

  - 活発なコミュニティ

- **学習コスト**: 高（Rust必要）

#### Wails（推奨度：中）
- **言語**: Go + Web技術
- **ライセンス**: MIT

  - 一部機能が開発中（マルチウィンドウ等）

#### NeutralinoJS（推奨度：中）
- **言語**: JavaScript/TypeScript
- **ライセンス**: MIT
- **特徴**:

  - 機能制限あり
- **商用利用**: 完全対応
- **学習コスト**: 低

### 2. 商用利用フレンドリーなライセンス

#### 最も商用利用に適したライセンス

**MIT License**:
- 最も制約が少ない
- 著作権表示のみ必要
- 商用利用・再配布・改変すべて自由
- 特許権の明示的保護なし

**Apache 2.0 License**:
- 商用利用フレンドリー
- 特許権の明示的許可
- 変更箇所の明示が必要
- NOTICE ファイルの保持が必要

**BSD License**:
- MIT類似の制約の少なさ
- 著作権表示のみ必要
- 商用利用・再配布・改変すべて自由

**EPL 2.0 (Eclipse Public License)**:
- 商用利用フレンドリー
- 弱いコピーレフト
- 特許権保護
- 商用ライセンス可能
